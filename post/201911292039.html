简述

RSA加密算法是一种非对称加密算法。在公开密钥加密和电子商业中RSA被广泛使用。

整个加密过程要用到p、q、n、L、E、D六个数据：

p、q：选取两个足够大的素数 p、q
n：令 n = p * q
Fn：Fn 是 n 的欧拉函数，（此处为（p-1）乘以（q-1））
e：选取一个e使得 1 < e < Fn，且 gcd(e，Fn) = 1
d：d 为 1 = e mod Fn 的乘法逆

此时，
（e，n）为公钥
（d，n）为私钥

加密过程：c = ( m ^ e ) % n

解密过程：m = ( c ^ d ) % n
